# wechat_info_collect
调查取证 | 针对微信客户端的信息收集工具, 一键提取本地PC所有的微信信息, 包括微信号, 手机号等  
暂且写个python版本的, 后续再写c或者其他语言的, 使用会更方便    
已经测试过部分系统, 但是在某些特殊情况下依然会有失败的可能, 这些情况下无法修复Bug, 如果实战遇到的话, 需要自己手动查看地址  


安装环境依赖
>pip3 install pypiwin32
